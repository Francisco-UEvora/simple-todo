# Simple TODO (Python)

Projecto base para exercício de introdução ao Git.

Objetivo do exercício (resumido):
- Clonar o repositório.
- Criar branches e fazer commits.
- Implementar suporte a tarefas urgentes (ver comentários TODO no código).
- Provocar e resolver um conflito (editar a constante WELCOME em main.py em branches diferentes).

Como usar:
```bash
python3 main.py list
python3 main.py add "Comprar pão"
python3 main.py done 1
python3 main.py remove 2
```
